📡 API مستقلة – استخدمها في بوتات واتساب وتيليجرام

أرسل طلب POST أو GET مع الكود ونوع التشويش لتحصل على النتيجة فورًا.

1. 🔐 تشويش عادي (Standard Obfuscation)

POST /api/obfuscate
{ "code": "console.log('test');", "options": "standard" }

2. ⚙️ تشويش متقدم (Advanced Obfuscation)

POST /api/obfuscate
{ "code": "console.log('test');", "options": "advanced" }

3. 🧠 تشويش VM (مثل الموقع الأصلي)

POST /api/vm-obfuscate
{ "code": "function test() { return 42; }" }

📌 الاختصارات (للاستخدام السريع في البوتات)

GET /api/obfuscate?code=console.log('hi')&options=standard
GET /api/vm-obfuscate?code=alert(1)
